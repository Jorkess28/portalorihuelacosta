# 🌴 Orihuela Costa Smart Portal 2026

![Status](https://img.shields.io/badge/Status-Live-success?style=for-the-badge&logo=github)
![Version](https://img.shields.io/badge/Version-2.6_Cyberpunk-blue?style=for-the-badge)

**Portal Comunitario Inteligente** diseñado para centralizar la información de empleo, seguridad y servicios de Orihuela Costa utilizando una interfaz futurista de alto rendimiento.

---

## 🚀 Características Principales

* **⚡ Sistema 1-Click Apply:** Perfil de candidato persistente mediante `LocalStorage`. Postula a empleos en segundos sin resubir documentos.
* **🚨 Terminal de Incidencias:** Sistema de reportes ciudadanos en tiempo real (tráfico, medusas, parkings) con estética de terminal de seguridad.
* **🌑 Protocolo Nocturno (Amber Mode):** Adaptación visual automática del espectro lumínico a partir de las 20:00 para reducir fatiga visual.
* **📡 Radar Live Ticker:** Teletipo de noticias dinámico con actualizaciones críticas de la zona.
* **📱 Diseño Ultra-Responsive:** Optimizado para dispositivos móviles y tablets para uso en exteriores (playas y comercios).

## 🛠️ Stack Tecnológico

* **Frontend:** HTML5, Tailwind CSS (JIT Mode).
* **Iconografía:** FontAwesome 6.0 (Pro-look).
* **Lógica:** JavaScript ES6+ (Arquitectura Modular).
* **Hosting:** GitHub Pages.
